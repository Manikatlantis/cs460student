RO.ME "3 DREAMS OF BLACK"

RO.ME is an incredible WebGL demo!. This WebGL website is an insight into 3 dreams, the dreams have been visualized as an interactive 3D world. It's like a music video made for the browser. The models are being rendered in real time depending on where the user looks to. RO.ME demo was created in 2011 and it was the first of its kind where a browser with WebGL was used to make immersive 3D graphics visible on our machines using our integrated graphics card. It showed that just with a simple tool like WebGL, such interactable and real things could come to life. This really opened the door for creativity when it comes to web design fusing audio, video, real-time rendering etc.

Technology: In this RO.ME demo, the primary technology used is HTML 5 combined with CSS for styling. Both these were handled under JavaScript and WebGL was the main highlight of this demo integrating audio, video and real time rendering really made this website a delight to go through!
